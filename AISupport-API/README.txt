To have the Flask library (does all the web stuff) in this folder do the following:
1
Install python (latest)
2
Install pip
3 
Install virtualenv
4
Command line: 
virtualenv venv
5
Command line: 
cd venv/Scripts
pip install flask

To run server:
Command line (in root)
./app.py or python ./app.py if doesnt work


Cheers!
